Lutest / terminal language

Quiet UI. Clear state.

The implemented terminal language for Lutest. Neutral surfaces carry the interface; color reinforces state, and symbols preserve it without color.

Palette

> context or section
+ completed action
x failed action
! user action needed
- compact metadata
Color reinforces, never replaces

Command surface

A compact map, not a manual page

Lutest Co-located tests for Luau modules.

  lutest test game --runtime roblox
